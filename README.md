# 📦 Bases de Datos II — TUIA

Repositorio personal con ejercicios prácticos y trabajos de la materia **Bases de Datos II** de la Tecnicatura Universitaria en Inteligencia Artificial (TUIA).

---

## 📚 Descripción

La materia aborda el diseño y explotación de bases de datos orientadas al análisis, cubriendo temas como:

- Data Warehouse y modelado dimensional
- Procesamiento OLAP
- ETL con SSIS
- Bases de datos NoSQL
- Business Intelligence y dashboards con Power BI
- Otros modelos: documentales, espaciales, temporales

---

## 🛠️ Tecnologías utilizadas

| Herramienta | Uso |
|---|---|
| Microsoft SQL Server Developer | Motor principal de base de datos |
| SQL Server Management Studio (SSMS) | Ejecución de consultas y administración |
| Visual Studio + SQL Server Data Tools | Desarrollo de paquetes ETL (SSIS) |
| Power BI Desktop | Reportes y dashboards |

---

## ▶️ Cómo correr los scripts

1. Instalar **SQL Server Developer Edition** y **SSMS**
2. Abrir el archivo `.sql` deseado desde SSMS
3. Seleccionar la base de datos correcta en el desplegable
4. Ejecutar con `F5`
